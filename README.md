# Cnidaria Evolutionary Omics

This repository contains computational workflows for evolutionary omics analyses in cnidarians.

## Current Workflows

### Bulk RNA-Seq Analysis
A Snakemake pipeline for standard bulk RNA-Seq data processing and differential expression analysis.

**Features:**
- Quality control (FastQC + MultiQC)
- Transcript quantification (Kallisto)
- Differential expression (DESeq2)
- Visualization
